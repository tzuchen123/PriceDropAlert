## db

### Redis
存歷史最低價，如果新價格低於最低價，更新mongodb且通知使用者

最佳做法是：
1.在應用程式(爬蟲)啟動時 建立 Redis 連線，並讓連線持續存在。
2.不要在每次查詢時才檢查是否已連線，這樣可以避免重複連接。
3.應用程式結束時（SIGINT）關閉 Redis 連線，確保資源釋放。
